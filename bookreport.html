Hello and welcome to my book report! Today I will be teaching you all HTML (Hyper Text Markup Language) and CSS (Cascading Style Sheets). The first thing that everyone should know is every HTML (NOT CSS) tag has a less than sign (<) and then the name of the tag (ex. "p") and then a greater than sign (>). The most used tag is "p". "p" is used when typing text to show on the website (I used it to make this paragraph). Speaking of paragraph who can guess what "p" stands for?? CORRECT!!! Are you wondering how to close the paragraph?? Add a slash before the tag (ex. <p>I LIKE TACOZ!!!!</p>).

I forgot to mention, in HTML you should always start your code with <!DOCTYPE html> and there is no ending tag. Some more things you should always have in your code are <html>, </html>, <head>, </head>, <body>, and </body>. Some more very useful tags are <h1>, <h2>, <h3>, <h4>, <h5>, and <h6>.
